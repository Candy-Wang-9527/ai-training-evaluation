## 🔐 安全性说明

### App Secret的安全使用

**❌ 旧的方式（不安全）：**
```javascript
// 前端代码中直接使用
const APP_SECRET = '1qkoA78XBoye66WmSRypuhHpIHt2Qlso'; // ❌ 暴露给所有用户
```

**✅ 新的方式（安全）：**
```javascript
// 后端代码中使用环境变量
const APP_SECRET = process.env.FEISHU_APP_SECRET; // ✅ 只有服务器能访问
```

### 环境变量的保护

**多层保护：**
1. ✅ .env文件已添加到.gitignore（不会提交到Git）
2. ✅ Vercel使用加密存储
3. ✅ 前端无法访问后端环境变量
4. ✅ 生产环境和开发环境分离
