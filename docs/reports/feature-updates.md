## 🆕 新增功能模块

### 1. 灵活的角色配置系统

#### 支持的角色类型
```javascript
SYSTEM_ROLES = {
    HR: 'hr',                    // 人力部门
    JUDGE: 'judge',              // 评审员
    INSTRUCTOR: 'instructor',    // 讲师
    GM: 'gm',                    // 总经理
    ADMIN: 'admin'               // 管理员
}
```

#### 角色权限说明
| 角色 | 主要职责 | 核心权限 |
|------|----------|----------|
| **人力部门** | 培训管理 | 创建培训、编辑信息、提交评审、管理参与人员 |
| **评审员** | 评分评估 | 查看分配任务、进行评分、提交评审、添加评语 |
| **讲师** | 培训实施 | 查看自己培训、管理培训内容、上传培训材料 |
| **总经理** | 最终审批 | 查看所有评估、批准评估、调整评分、系统配置 |
| **管理员** | 系统维护 | 配置角色、管理系统、管理用户、导出数据 |

#### 三种配置方式
1. **按姓名配置**: 通过员工姓名直接分配角色
2. **按用户ID配置**: 通过飞书用户ID精确分配
3. **按部门配置**: 整个部门的人员统一分配角色

#### 核心文件
- **js/role-config.js** - 角色配置管理模块
- **role-config.html** - 角色配置管理界面

### 2. 培训基础信息录入页面

#### 页面位置
- **文件名**: `training-input.html`
- **访问路径**: 导航栏 → 培训录入

#### 主要功能
1. **基础信息录入**
   - 培训名称、类型
   - 培训开始/结束日期
   - 培训地点和描述

2. **人员信息管理**
   - 选择培训讲师
   - 添加参与人员
   - 按部门批量添加人员

3. **培训内容设置**
   - 培训时长、人数统计
   - 培训目标和大纲

4. **草稿保存**
   - 支持保存草稿功能
   - 自动保存最近录入记录

#### 权限控制
- ✅ **人力部门**: 可以创建和编辑培训信息
- ❌ **其他角色**: 无权限访问

#### 表单字段

### 3. 角色配置管理页面

#### 页面位置
- **文件名**: `role-config.html`
- **访问路径**: 导航栏 → 角色配置

#### 主要功能

   - 不影响实际配置

6. **配置导入导出**
   - 导出为JSON文件备份
   - 从文件导入配置

## 💡 使用指南

### 人力部门使用流程

#### 1. 配置角色（管理员）
1. 访问 `role-config.html`
2. 添加用户角色配置：
   - 选择"按姓名配置"
   - 输入员工姓名（如"张三"）
   - 分配角色为"人力部门"
3. 确认添加

#### 2. 录入培训信息
1. 访问 `training-input.html`
2. 填写培训基础信息：
   - 培训名称："2026年Q1 AI工具培训"
   - 培训类型："入门培训"
   - 培训日期：选择开始和结束日期
3. 选择培训讲师（从讲师角色中选择）
4. 添加参与人员：
   - 点击"添加参与人员"逐个选择
   - 或点击"按部门批量添加"
5. 填写培训内容、目标等
6. 点击"提交培训信息"

### 评审员使用流程

#### 1. 配置评审员角色
- 将相关员工配置为"评审员"角色

#### 2. 进行评分
1. 访问 `index.html`
2. 选择待评估员工
3. 对7个维度进行评分
4. 提交评审结果

### 管理员配置流程

#### 1. 批量配置角色
```javascript
// 方法1: 通过角色配置页面
访问 role-config.html → 添加用户角色配置 → 逐个添加

// 方法2: 通过导入配置
1. 创建JSON配置文件
2. 在角色配置页面点击"导入角色配置"
3. 选择文件导入
```

#### 2. 导出配置备份
```javascript
在角色配置页面 → 点击"导出角色配置" → 保存JSON文件
```

## 📞 使用支持

### 常见问题

**Q: 如何批量配置用户角色？**
A: 可以通过角色配置页面的导入功能，创建JSON配置文件后批量导入。

**Q: 培训信息录入后如何修改？**
A: 目前需要联系人力部门或管理员，后续版本将增加编辑功能。

**Q: 角色配置会同步到飞书吗？**
A: 不会，角色配置完全独立存储在本地，不影响飞书组织架构。

**Q: 如何测试不同角色的功能？**
A: 在角色配置页面的"测试角色"区域选择要测试的角色，点击"设置测试角色"即可。
