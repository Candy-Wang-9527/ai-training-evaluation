## 🗑️ 一键删除命令

### Windows PowerShell
```powershell
# 备份文件
Remove-Item "config.html.backup"
Remove-Item "index.html.backup"
Remove-Item "js\app.js.backup"

# 临时文件
Remove-Item "index_fixed.html"
Remove-Item "js\app_fixed.js"

# 旧测试文件
Remove-Item "diagnostic_test.html"
Remove-Item "test.html"
Remove-Item "test_verification.html"

# 过时文档
Remove-Item "修复说明.md"
Remove-Item "问题解决指南.md"
```

### Linux/Mac Terminal
```bash
# 备份文件
rm config.html.backup
rm index.html.backup
rm js/app.js.backup

# 临时文件
rm index_fixed.html
rm js/app_fixed.js

# 旧测试文件
rm diagnostic_test.html
rm test.html
rm test_verification.html

# 过时文档
rm 修复说明.md
rm 问题解决指南.md
```

### 或使用通配符一次性删除
```bash
# Windows PowerShell
Remove-Item *.backup
Remove-Item *_fixed.*
Remove-Item diagnostic_test.html,test.html,test_verification.html
Remove-Item 修复说明.md,问题解决指南.md

# Linux/Mac
rm *.backup
rm *_fixed.*
rm diagnostic_test.html test.html test_verification.html
rm 修复说明.md 问题解决指南.md
```
