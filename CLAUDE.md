## 📋 项目概述

这是一个用于评估员工AI培训和应用效果的飞书集成系统。用户可以在飞书工作台中使用此应用，为员工进行培训评分和应用效果评估，数据自动同步到飞书多维表格（Base）。

### 核心功能
1. **评分模块** - 对员工进行AI培训和应用效果评分
2. **培训录入** - 录入培训基础信息（需要HR权限）
3. **统计分析** - 查看评分统计数据和趋势
4. **角色配置** - 配置用户角色和权限
5. **系统配置** - 配置评估权重、飞书Base连接等

## 🏗️ 技术架构

### 前端
- **框架：** 纯JavaScript（无框架）
- **UI库：** Bootstrap 5.3.0
- **图标：** Font Awesome 6.4.0
- **图表：** Chart.js 4.3.0

### 后端
- **框架：** Node.js + Express
- **部署：** Vercel Serverless Functions
- **API设计：** RESTful API

### 飞书集成
- **前端SDK：** 飞书JS SDK（lark-jsapi）
- **后端API：** 飞书服务器端API
- **数据存储：** 飞书多维表格（Base）

## ⚙️ 环境配置

### Vercel环境变量
```bash
FEISHU_APP_ID=cli_a123...              # 飞书应用ID
FEISHU_APP_SECRET=xxx...               # 飞书应用密钥
FEISHU_BASE_APP_TOKEN=xxx...           # Base应用Token（可选）
FEISHU_BASE_TABLE_ID=xxx...            # Base表格ID（可选）
```

### 飞书应用权限
**必需权限：**
- `contact:department:readonly` - 查看部门信息
- `contact:user:readonly` - 查看用户信息
- `contact:user.base:readonly` - 查看用户基本信息
- `bitable:app:readonly` - 读取多维表格
- `bitable:app:readwrite` - 读写多维表格
